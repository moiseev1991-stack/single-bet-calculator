How it works

A single bet is one selection in any market or sport. Enter your stake and odds — return and profit update in real time. Enable each-way to split into a win bet and a place bet, or add a Rule 4 deduction for withdrawn runners.

How do I use the Single Bet Calculator correctly?

Enter your stake and pick your odds format (fractional, decimal, or American). The calculator shows total return and profit for a single selection in real time. For UK horse racing, enable each-way and apply any Rule 4 deduction to the winnings portion. Always follow the settlement terms shown on your bet slip.

Which odds formats can I choose: fractional, decimal, American?

The calculator accepts all common UK and international formats. Type fractional odds like 7/4, decimal like 2.75, or American like +175 or -150. The tool converts between formats and uses the correct payout logic for each.

  • Fractional (a/b): Profit = stake × (a/b); Return = stake + profit.
  • Decimal (D): Profit = stake × (D − 1); Return = stake × D.
  • American positive (+X): Profit = stake × (X / 100); Return = stake + profit.
  • American negative (−Y): Profit = stake × (100 / Y); Return = stake + profit.

How do I enter stake, each-way, and Rule 4 deductions?

Follow what is printed on your bet slip. For each-way, enter the place terms (place fraction and number of places). For Rule 4, enter the deduction value announced by the bookmaker. The calculator applies the deduction to the winnings portion only.

  • If each-way: input place fraction (e.g., 1/5 or 1/4) and number of places in Settings.
  • If Rule 4: input the announced deduction (e.g., 25%). The calculator reduces only the winnings.

What Is a Single Bet?

A single bet is the most straightforward wager in sports betting — one selection, one outcome, one result. If your selection wins, you collect your stake multiplied by the decimal odds. Unlike an accumulator, a single is not dependent on any other selection, so there is no risk of a chain of results going wrong.

Using a single bet calculator removes the need for mental arithmetic, particularly when dealing with fractional odds or applying Rule 4 deductions. The calculator also handles each-way versions, where your stake is doubled to cover both a win bet and a place bet on the same selection.

How are return, profit, and implied probability worked out?

For a single bet, profit is the winnings excluding your original stake, return is profit plus stake, and implied probability is the probability the odds encode. Decimal odds already include the stake in the price, while fractional odds state the profit relative to the stake.

Decimal vs fractional: what changes in the maths?

  • Decimal (D): Profit = stake × (D − 1); Return = stake × D; IP = 1 / D.
  • Fractional (a/b): Profit = stake × (a/b); Return = stake + profit; IP = b / (a + b).
  • Conversion: D = 1 + (a / b).

Single bet example: £10 at 7/4 step-by-step

  • Convert: 7/4 in decimal is 1 + 7/4 = 2.75.
  • Profit: £10 × (7 ÷ 4) = £17.50.
  • Return: £17.50 + £10.00 = £27.50.
  • Implied probability: 4 / 11 ≈ 36.36%.
  • If wins: £27.50. If loses: £0.00. If void: £10.00 stake returned.

Will my results match major UK bookmakers?

Yes — provided you use the same odds, settlement rules, and deductions, your profit and return should match to the penny. Small discrepancies can occur when a bookmaker rounds at intermediate steps. Always check the official rules published by your bookmaker (e.g., William Hill).

Single Bet Each Way — How It Works

An each-way single is two bets at the same stake: a win bet and a place bet. If your selection wins, both parts return — the win bet at full odds, the place bet at a fraction of the win odds (typically 1/4 or 1/5). Toggle Each Way in the calculator above, then configure place fraction and number of places in Settings.

Single vs other bet types: doubles, forecasts, and accas

A single bet settles on one selection, so your stake faces one outcome and one overround. Multiples like doubles and accumulators compound prices and risk across legs: the potential profit rises, but so does variance and exposure to bookmaker margin.

When is a single better than a double?

  • Small edge, big variance: a double dilutes EV when one leg is fair value.
  • Independent legs: combined true-win probability is p(A) × p(B) — both must win.
  • Settlement friction: voids, Rule 4s on one leg can erode the headline double payout.
  • Bankroll management: singles allow grading confidence by staking per selection.

Worked EV comparison:

Single on A at 6/4 (2.50), true p(A) = 0.45, stake £10 → EV +£1.25

Double with B at 5/6 (1.83), true p(B) = 0.52, same stake → EV +£0.71

Do you even need a calculator for a single bet?

For decimal odds with clean stakes, mental maths is fast. But Rule 4 deductions, each-way place fractions, dead-heats, or free-bet SNR terms can change the payout by pounds. A calculator standardises rounding, applies deductions correctly, and mirrors UK settlement rules.

Edge cases where errors creep in

Rule 4 (applies to winnings only)

£10 at 7/2 (4.50). Winnings = £35.00. With 25p Rule 4: deduction £8.75; adjusted profit £26.25; return £36.25.

Each-way place terms

£10 EW at 8/1, place 1/5. Win return £90. Place price = 8/5 (2.60), profit £16, return £26. Total if wins: £116.

Dead-heat

Two-way dead-heat, £10 at 6/1. Effective stake = £5. Profit £30; Return £35.

Free bet SNR

£10 free bet at 2.75. Return = £17.50 (stake not included), not £27.50.
